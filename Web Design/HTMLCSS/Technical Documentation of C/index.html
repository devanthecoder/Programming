Technical Documentation of the C Language

Introduction

C is a powerful general-purpose programming language. It is widely used for system programming, developing operating systems, embedded systems, and as a foundation for learning other languages. Developed in the early 1970s by Dennis Ritchie at Bell Labs, C remains a critical tool in modern computing.

Your First Program

Open your text editor or IDE. Write the following code and save it to a '.c' file of any name:

#include <stdio.h>   int main(){     printf("Hello world!");     return 0; }

Compile it using your compiler and run the executable with the same name of the .c file.

Structure of a C Program

A typical C program includes:

  • Preprocessor directives: Begin with # (e.g., #include <stdio.h>).
  • Main function: The entry point (int main() {}).
  • Statements: End with a semicolon (;).

Example:

#include <stdio.h>   int main() {     printf("This is a basic structure of a C program.");     return 0; }

This program demonstrates the minimum components required for a C program to compile and run.

Data Types

Data types in C can be divided into two broad classes:

  • Primitive types: int, float, char, double, etc.
  • Derived types: Arrays, pointers, structures.
Variables and Constants

  • Variables: Declared with a type.

Example:

int a = 10; float b = 3.14; char c = 'C';

This code declares variables of different primitive data types and initializes them with values.

  • Constants: Declared with a type and const keyword.

const int maxUsers = 100;

The maxUsers variable is a constant, meaning its value cannot change during program execution.

Operators

Operators can be broadly classfied into many types like:

  • Arithmetic Operators: Addition (+), Subtraction (-), Multiplication (*), Division (/) and Modulus (%).

These operators perform arithmetic operations on numbers and return their values.

int a=10, b=3; printf("Addition: %d\n",a+b); //Prints the sum of a and b, which is 13. printf("Subtraction: %d\n",a-b); //Prints the difference of a and b, which is 7. printf("Multiplication: %d\n",a*b); //Prints the product of a and b, which is 30. printf("Division: %d\n",a/b); //Prints the quotient of a divided by b, which is 3. printf("Modulus: %d\n",a%b); //Prints the remainder of a divided by b, which is 1.

This code snippet performs various arithmetic operations between two integers a and b and prints their results.

  • Relational Operators: Equal to (==), Not Equal to (!=), Less Than (<), Greater Than (>), Less Than or Equal to<=, Greater Than or Equal to (>=).
  • Logical Operators: Logical NOT (!), Logical AND (&&), Logical OR (||).

Statements consisting of these operators return 1 if they are True and 0 if False.

int a=5, b=10; printf("Inequality: %d\n",a!=b); //Prints 1 as a is not equal to b. printf("Comparison: %d\n",a>b); //Prints 0 as a is not greater than b. printf("Logical NOT: %d\n",!(a!=b)); //As a!=b returns 1, !(a!=b) will print 0. printf("Logical OR: %d\n",(a!=b)||(a>b)); //As a!=b and a>b return 1 and 0 respectively, the statement here will print 1. printf("Logical AND: %d\n",(a!=b)&&(a>b)); //As a!=b and a>b return 1 and 0 respectively, the statement here will print 0.

This code evaluates logical and relational expressions between two integers/statements and prints their results.

  • Bitwise Operators: Bitwise NOT (~), Bitwise AND (&), Bitwise OR (|), Bitwise XOR (^), Right Shift (>>), Left Shift (^).
int a=5, b=10; //5=0101, 10=1010 printf("Bitwise NOT: %d\n",~a); //Prints -6 by performing bitwise NOT on a. printf("Bitwise OR: %d\n",a|b); //Prints 15 by performing bitwise OR on a and b. printf("Bitwise AND: %d\n",a&b); //Prints 10 by performing bitwise AND on a and b. printf("Bitwise XOR: %d\n",a|b); //Prints 15 by performing bitwise XOR on a and b. printf("Right Shift: %d\n",b>>2); //Prints 2 by shifting the bits of b two places to the right. printf("Bitwise OR: %d\n",b<<a); //Prints 320 by shifting the bits of b 'a' places to the left and filling the empty places with 0s.

This code evaluates logical and relational expressions between two integers/statements and prints their results.

  • Assignment Operators: Simple:=, Compound: += -=, *=, /=, %=.

These operators assign/reassign values to variables.

int a=5; //Variable a is assigned the value of 5. a+=7; //a is now reassigned to 12. a-=2; //a is now reassigned to 10. a*=3; //a is now reassigned to 30. a/=6; //a is now reassigned to 5. a%=4; //a is now reassigned to 1.

Using assignment operators, this code assigns/reassigns variable a to various values.

Conditional Statements

Conditional Statements in C come in 3 main categories:

  • Ternary/Conditional Operator (?:):The Ternary Operator evaluates a condition and returns a value depending on whether the condition is True or False.

int a, b; scanf("%d %d", &a, &b); //Inputting values to a and b printf("The greater number is %d.", a>b?a:b);

Here, the code takes two inputs from the user and returns the greater number using Ternary Operator.

  • If - Else If - Else: These conditional statements work as such:

    • The condition of the if block is evaluated first. If the condition is true, then the statement within the if block is executed. If false, then the next condition is checked.
    • Then the condition of the subsequentelse if block is evaluated next. If the condition is True, then the statement within the else if block is executed. If false, then the next condition is checked.
    • If all the if and else if conditions are false, the statements within the else block are executed as a fallback.

int a; scanf("%d", &a); if(a>0){     printf("%d is a positive number.", a); }else if(a<0){     printf("%d is a negative number.", a); }else{     printf("The number is zero."); }

Here, the code takes an integral input from the users and checks whether that inputted number is positive, negative or zero using if - else if - else statements.

  • Switch - Case: These conditional statements work as such:

    • The expression within the switch block is evaluated and then compared with the value in each case.
    • If the value of a case matches with that of the expression in the switch block, then all the statements under that case block are executed.
    • If none of the values of in the cases match with that of the expression in the switch block, then all the statements under the default block are executed.

char grade; scanf("%c", &grade); switch(grade){     case 'A';     printf("Amazing grade! You pass with flying colors!");     break;     case 'B';     printf("Could be better, but you still did decent!");     break;     case 'C';     printf("Meh. Average.");     break;     case 'D';     printf("Almost failed there bucko.");     break;     case 'FR';     printf("Utter failure");     break;     default;     printf("Invalid grade"); }

Here, the code takes an character input from the user and prints an appropriate output using switch - case statements.

Loops

In C, Loops come in 3 types:

  • For Loops: for loops iterate over a set of statements for a fixed number of times specified by the user. The for loop declaration consists of three parts:

    • Initialization: The iterating variable is initialized to some value.
    • Condition: The loop iterates till the specified condition becomes false.
    • Change: After each iteration, the iterating variable changes in value.
for(int i = 0; i < 5; i++){     printf("Number at current iteration: %d", i); }

This code snippet initializes iterating variable i and prints its value for each iteration of the for loop.

  • While Loops: while loops iterate over a set of statements till the condition within the loop declaration becomes false.

    In both while and for loops, the conditions are checked before execution of the statements within those loops.

int a = 1; while(a < 5){     printf("Iterating statement number %d", a);     a++; }

Here, the code initializes a variable a and its value is printed for each iteration of the while loop.

  • Do - While Loops: do-while loops are very similar to while loops barring one difference.

    The statements within the loop are executed before the condition of the loop is checked.

int a = 1; do{     printf("Iterating statement number %d", a);     a++; }while(a < 5)

Here, the code initializes a variable a and its value is printed for each iteration of the do-while loop.

Functions

A function is a block of code that performs a specific task. It takes input (optional), performs an operation, and returns a result (optional).

Defining Functions: Functions in C are defined as such:

  • Function Name and Type: The function's type and name are first defined in the function.
  • Function Inputs: The function's inputs and their corresponding types are then defined as parameters to the function.
  • Function Processing and Output: The statements within the function are executed and then the resultant value is returned with the return keyword.

As an example:

int add(int a, int b){     int sum = a + b;     return sum; }

The add function returns the sum of two inputs a and b.

Passing Parameters to Functions: Parameters can be passed to functions in two ways:

  • Pass by Value: Copies of the arguments are passed, and modifications don't affect the original variables. (like the add function above)
  • Pass by Reference: The addresses of the arguments are passed as inputs to the function, and any modifications to the inputs affect the original variables.

As an example:

void add2(int* a){     (*a) += 2; }

The add2 function increments the value of a by 2, which is only possible using Pass by Reference.

Pointers

Pointers are variables that store the memory addresses of other variables.

For example:

int x = 5; int* ptr = &x; printf("Address of x: %p\n", ptr); printf("Value of x: %p\n", *ptr);

Here, the ptr variable stores the address of variable x. The value stored at that address can be accessed using the dereference operator (*).

Pointer Arithmetic: As arrays are a contiguous sequence of values stored in memory, with a pointer to an element of the array, we can access its other elements by incrementing or decrementing the pointer.

int arr[] = {1, 2, 3}; int* ptr = arr; for(int i = 0; i < 3; i++){     printf("Element at index %d of the array is %d",i,*(ptr+i)); }

In this code snippet, we print the elements of the array using Pointer Arithmetic.

Memory Allocation

Contained in the <stdlib.h> library, C provides functions for memory management:

  • malloc and calloc: These functions allocates memory of user-specified size to a pointer dynamically at runtime.

  • realloc: This function reallocates memory of user-specified size to a pointer with previously allocated memory, essentially giving the user a way to resize a memory block dynamically.

  • free: This function deallocates previously allocated memory of a pointer and releases it back to the operating system at runtime.

int *arr = (int *)calloc(5, sizeof(int)); if (arr==NULL){     printf("Memory Allocation Failed");     return 0; }   for(int i=0; i<5; i++){     *(arr + i) = i*2;     printf("arr[%d] = %d\n", i, *(arr + i)); }   arr = (int *)realloc(arr, 6 * sizeof(int)); arr[5] = 10; printf("arr[%d] = %d\n", 5, arr[5]); free(arr);

This program dynamically allocates memory for an array of integers, assigns values to the array, reallocates memory to allow an extra element, and frees the allocated memory to avoid memory leaks.

File I/O

C provides functions to handle file input and output, including:

  • fopen: This function opens a file for reading, writing and/or appending(depending on the mode) by returning a pointer to the FILE object if successful or a NULL pointer otherwise.

  • fclose: This function closes a FILE object opened with fopen.

  • fprintf:This function is used to write formatted output to a file.

  • fscanf:This function is used to read formatted input from a file.

FILE *file = fopen("example.txt", "w"); if (file){     fprintf(file, "Hello, File!\n");     fclose(file); }   file = fopen("example.txt", "r"); if (file){     char buffer[50];     fscanf(file, "%s", buffer);     printf("Content: %s\n", buffer);     fclose(file); }

This code creates a file example.txt, writes a string to it, then reads the string and prints it to the console.

Structures, Unions and Enums

  • Structures: Structures group variables of different types under one name. Each variable within a structure is called a member of the structure.

Example:

struct Point {     int x;     int y; };   int main() {     struct Point p1 = {10, 20};     printf("Point: (%d, %d)\n", p1.x, p1.y);     return 0; }

This program defines a struct named Point with two integer members, x and y. An instance of Point is created and its values are printed.

  • Unions: Unions are similar to structures but use shared memory for their members and hence, can only store one member's value, setting the others' to 0 or NULL.

Example:

union Data {     int i;     float f; };   int main() {     union Data d;     d.i = 10;     printf("Integer: %d\n", d.i);     d.f = 5.5     printf("Float: %d\n", d.f);     return 0; }

In this program, the union Data can hold an integer or a float, but not both simultaneously due to shared memory.

  • Enums: Enums define a set of named integer constants..

Example:

enum Day {Sunday, Monday, Tuesday};   int main() {     enum Day today = Monday;     printf("Today is: %d\n", today);     return 0; }

The enum Day defines constants representing days of the week. The integer value of Monday is assigned to today.
